## 📊 看板使用规范

### 1. 企业知识库(主要维护区)
**访问权限**: 完全控制

#### 知识库分类结构
```
📚 企业知识库 (主页)
├── 📋 项目文档
│   ├── 需求文档
│   ├── 设计文档
│   ├── 技术方案
│   └── 会议纪要
├── 📖 规范标准
│   ├── 开发规范
│   ├── 设计规范
│   ├── 测试规范
│   └── 运维手册
├── 🚀 迭代归档
│   ├── 2024-Q1
│   ├── 2024-Q2
│   └── ... (按季度归档)
├── 🐛 Bug知识库
│   ├── 经典Bug案例
│   ├── 问题排查手册
│   └── 解决方案库
├── 📦 交付产物
│   ├── API文档
│   ├── 用户手册
│   ├── 部署文档
│   └── 培训资料
└── 🔗 外部资源
    ├── 第三方文档
    ├── 合作方资料
    └── 行业参考
```

#### 文档入库标准

| 字段 | 说明 | 必填 |
|------|------|------|
| 文档标题 | 清晰描述文档内容 | ✓ |
| 文档类型 | 需求/设计/技术/会议/规范/其他 | ✓ |
| 所属项目 | 关联项目或迭代 | ✓ |
| 创建时间 | 文档创建日期 | ✓ |
| 最后更新 | 文档更新时间 | 自动 |
| 作者 | 文档撰写人 | ✓ |
| 标签 | 关键词标签(多选) | ✓ |
| 状态 | 草稿/审核中/已发布/已归档 | ✓ |
| 访问权限 | 公开/内部/机密 | ✓ |
| 关联迭代 | 链接到迭代看板 | - |
| 版本号 | 文档版本 | 规范/API文档必填 |
| 审核人 | 审核通过的负责人 | 已发布状态必填 |

#### 归档规范

**交付产物归档时机**:
- 迭代任务标记"已完成"后24小时内
- 产品版本发布后一周内
- 项目里程碑达成后

**归档检查清单**:
- [ ] 文档格式统一(PDF/MD/HTML)
- [ ] 版本号已标注
- [ ] 关联迭代已链接
- [ ] 标签已打全
- [ ] 访问权限已设置

### 2. Bug看板(与测试共同维护)
**访问权限**: 修改状态、更新进度、归档

#### Bug看板字段维护

| 字段 | 陆贾维护动作 | 说明 |
|------|--------------|------|
| 状态 | ✓ 更新流转 | 确保状态准确反映当前进度 |
| 修复版本 | ✓ 填写 | Bug修复所在的版本号 |
| 验证结果 | ✓ 确认 | 测试验证后更新 |
| 知识库关联 | ✓ 链接 | 有价值的Bug关联到Bug知识库 |
| 归档标记 | ✓ 标记 | 已关闭且超过30天的Bug归档 |
| 重复标记 | ✓ 标注 | 标记重复Bug并链接原Bug |

#### 11:00同步流程
```
1. 登录Notion Bug看板
2. 检查测试更新的Bug状态
3. 确认"已修复"状态的Bug
4. 更新"待验证"状态并通知测试
5. 将"已关闭"超过30天的Bug移至归档数据库
6. 记录当日数据: 新建/修复/关闭数量
```

### 3. Notion与OpenViking同步

#### 同步范围
| Notion内容 | OpenViking目标 | 同步频率 |
|------------|----------------|----------|
| 知识库文档 | 文档中心 | 实时/每日 |
| Bug数据 | 缺陷管理系统 | 每日 |
| 迭代交付物 | 版本库 | 迭代结束时 |
| 会议纪要 | 协作平台 | 即时 |

#### 同步检查清单
- [ ] 检查Notion中"已发布"状态的文档
- [ ] 确认OpenViking对应模块是否已更新
- [ ] 验证链接有效性
- [ ] 处理同步冲突

## 📝 归档操作规范

### 迭代交付产物归档流程

```
迭代结束
    ↓
[张良确认所有任务状态]
    ↓
收集交付产物列表
    ↓
按类型分类整理
    ↓
上传到Notion「交付产物」库
    ↓
填写归档信息(版本、日期、关联迭代)
    ↓
同步到OpenViking
    ↓
通知相关方归档完成
```

### 交付产物数据库字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 产物名称 | 交付物名称 | API v2.0 文档 |
| 产物类型 | 文档/代码/配置/其他 | 文档 |
| 所属迭代 | 关联迭代日期 | 2024-03-25 |
| 版本号 | 版本标识 | v2.0.1 |
| 交付时间 | 实际交付日期 | 2024-03-25 |
| 存储位置 | 链接或路径 | [Notion链接] |
| 接收方 | 谁接收了这个交付物 | 客户A/运维团队 |
| 归档状态 | 待归档/已归档 | 已归档 |
| 备注 | 其他说明 | - |
