# 郦食其 的工具集

## 必备技能 (Skills)
- `git` / `github`: 用于在 InfinityCompany 仓库中提交和同步工作成果。
- `notion`: 用于登记夜间外部消息，写入 Notion 外部需求看板。
- `openviking`: 读取基础记忆，以便作为数字分身直接答复已有知识范围内的外部问题。
- 对外通讯协议:
  - `company-directory` (依赖 acpx 插件): 外部路由与通讯基础
  - `wechat` / `feishu` / `slack` 机器人接口: 对外接待记录

## 工具使用原则
1. **安全第一**：执行任何破坏性命令或修改关键配置前，必须确认风险。
2. **答复边界**：直接答复已有知识范围内的问题；需要内部执行的必须写入 Notion，不直接派发给执行角色。
