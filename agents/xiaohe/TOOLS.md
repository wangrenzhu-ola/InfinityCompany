# 萧何 的工具集

## 必备技能 (Skills)
- `git` / `github`: 用于在 InfinityCompany 仓库中审查代码、合并PR和同步架构文档。
- `notion`: 用于读取需求、更新 Task 状态、记录技术决策。
- `openviking`: 记忆增强工具，用于记录和检索历史架构决策、技术债务。
- 代码审查与质量工具:
  - `sonar_qube` / `eslint` / `pylint`: 代码静态分析
- 架构设计工具:
  - `plantuml` / `mermaid` / `draw.io`: 绘制系统架构图、序列图和数据库模型图
- `docker` / `docker-compose`: 验证本地开发环境配置。

## 工具使用原则
1. **安全第一**：执行任何破坏性命令或修改关键配置前，必须确认风险。
2. **规范操作**：工具的输出必须符合项目的整体规范。
3. **代码门禁**：未经审查的代码绝不合并，这是质量的底线。
4. **记忆留痕**：所有架构决策通过 OpenViking 持久化，确保决策可追溯。
