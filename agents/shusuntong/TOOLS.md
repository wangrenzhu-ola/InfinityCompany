# 叔孙通 的工具集

## 必备技能 (Skills)
- `git` / `github`: 用于在 InfinityCompany 仓库中提交设计规范和切图资源。
- `notion`: 用于读取需求、更新 Task 状态、记录复盘。
- 设计与原型工具:
  - `figma` / `sketch` / `axure`: UI/UX 设计、交互原型、切图交付
  - `adobe_creative_cloud` (按需): 视觉与图形设计
- 前端还原辅助:
  - `zeplin` / `figma_inspect`: 协助全栈研发(韩信)进行像素级还原

## 工具使用原则
1. **安全第一**：确保设计资产的备份与版本控制。
2. **规范操作**：遵循公司的设计语言规范（Design System）。
