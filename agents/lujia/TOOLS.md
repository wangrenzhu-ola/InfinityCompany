# 陆贾 的工具集

## 必备技能 (Skills)
- `git` / `github`: 用于在 InfinityCompany 仓库中维护知识库结构定义、快照脚本。
- `notion`: 用于维护迭代看板、外部需求看板、Bug 看板。
- `openviking`: 安装到 OpenClaw 的记忆增强 skill，维护双层知识库。
- 文档处理工具:
  - `markdown_lint` / `mkdocs` / `docusaurus`: 沉淀SOP和规范文档
- 脚本执行:
  - `bash` / `python`: 执行 Notion 与 OpenViking 的同步脚本与资源导入脚本

## 工具使用原则
1. **安全第一**：执行任何破坏性命令或修改关键配置前，必须确认风险。
2. **规范操作**：每日 11:00 校验 Notion 与 OpenViking 同步状态。
3. **双层同步**：确保 Notion 主库与 OpenViking 主记忆的数据一致性。
